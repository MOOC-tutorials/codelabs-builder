  1. Tener aprovisionada su máquina virtual para el curso ISIS2603. Si aún no lo ha hecho, por favor, vaya al curso en sicua y siga las instrucciones para obtener su máquina virtual y para acceder a ella.

Temas a tratar

  • Clonar el repositorio
  • Abrir los proyectos

Organización github del curso

En github existe la organización https://github.com/Uniandes-isis2603 donde se encuentran creados los grupos (teams) y los repositorios de los proyectos de cada equipo.

El repositorio de su grupo para el proyecto front es 202010_SX_EY_nombreProyecto, donde X es el número de su sección, Y es el número de su equipo y nombreProyecto es el nombre abreviado de su proyecto.

El repositorio para el proyecto de su equipo es privado y solo puede ser visto por los integrantes del equipo que hayan aceptado la invitación.

Si no puede ver su repositorio, por la razón que sea, y mientras resuelve el problema, puede realizar este tutorial con el repositorio front-pregrado201920, que está en la misma organización y es público; de este modo podrá probar el ambiente de su máquina.

Instrucciones básicas

Para clonar un repositorio de github, es decir crear una copia en su máquina del repositorio en github y guardar información para su posterior sincronización, ud debe:

  1. Estar logueado en la máquina virtual con el usuario estudiante
  2. Estar ubicado en una carpeta donde va a crear la copia del repositorio.
  3. La carpeta debe estar ubicada en el directorio raíz en una ruta que no contenga espacios
  4. Tener la url del repositorio front que va a clonar

URL del repo que se va a clonar

En github.com localice el repositorio que va a clonar, seleccione "Clone or download" y copie la url.

Clonar desde línea de comandos

Desde una línea de comandos vaya a la carpeta de su preferencia e ingrese el comando

git clone url

donde url corresponde a la dirección del repositorio (por ejemplo, https://github.com/Uniandes-isis2603/202010_S3_E1_VecindarioAmigo_Front.git)

Luego aparecerá una ventana emergente en la que se le pedirán las credenciales de github.

Para abrir el proyecto se sugiere tener instalado Visual Studio Code, el cual lo puede descargar desde esta dirección: https://code.visualstudio.com/

Una vez haya instalado Visual Studio Code, abra la carpeta en donde clonó el proyecto.

En Visual Studio Code, abra una nueva terminal y asegúrese de que está en la raíz de su proyecto. Luego, ingrese el comando npm install. Esto instalará las dependencias necesarias.

Para ejecutar la aplicación, desde la misma terminal e ingrese el comando ng serve

Vaya a un navegador e ingrese la dirección http://localhost:4200
